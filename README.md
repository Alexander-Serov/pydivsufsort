# `pydivsufsort`: bindings to libdivsufsort

`pydivsufsort` prebuilds `libdivsufsort` as a shared library and
includes it in a Python package with bindings.

## Installation

On Linux and macOS:
```
pip install pydivsufsort
```

Untested on Windows.

## Testing

```
pytest
```
